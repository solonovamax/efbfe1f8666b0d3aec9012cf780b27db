Package-level declarations

The Networking API.

For login stage networking see org.quiltmc.qsl.networking.api.ServerLoginNetworking. For play stage networking see org.quiltmc.qsl.networking.api.ServerPlayNetworking.

For events related to the connection to a client see org.quiltmc.qsl.networking.api.ServerLoginConnectionEvents for login stage or org.quiltmc.qsl.networking.api.ServerPlayConnectionEvents for play stage.

For events related to the ability of a client to receive packets on a channel of a specific name see org.quiltmc.qsl.networking.api.S2CPlayChannelEvents.
