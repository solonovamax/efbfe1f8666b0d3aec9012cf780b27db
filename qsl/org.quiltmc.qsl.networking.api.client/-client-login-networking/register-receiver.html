open fun registerReceiver(channelName: Identifier, queryHandler: ClientLoginNetworking.QueryRequestReceiver): Boolean

Registers a handler to a query request channel.

If a handler is already registered to the channelName, this method will return false, and no change will be made. Use unregisterReceiver to unregister the existing handler.

Return

false if a handler is already registered to the channel name, otherwise true

channelName

the identifier of the channel
