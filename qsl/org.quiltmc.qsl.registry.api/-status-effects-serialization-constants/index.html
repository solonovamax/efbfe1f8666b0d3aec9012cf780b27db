StatusEffectsSerializationConstants

class StatusEffectsSerializationConstants

Represents StatusEffects serialization keys.

Quilt Registry patches some serialization methods to use full identifiers instead of raw numerical identifiers as those can cause issues upon world save in a modded context. Raw identifiers are still saved for Vanilla parity.
