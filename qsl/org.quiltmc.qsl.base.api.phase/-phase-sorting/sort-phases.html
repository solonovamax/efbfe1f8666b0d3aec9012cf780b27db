sortPhases

open fun <T, P : PhaseData<T, P>?> sortPhases(sortedPhases: List<P>)

Deterministically sort a list of phases. 1) Compute phase SCCs (i.e. cycles). 2) Sort phases by id within SCCs. 3) Sort SCCs with respect to each other by respecting constraints, and by id in case of a tie.

Parameters

sortedPhases

the phases to sort

<T>

<P>
