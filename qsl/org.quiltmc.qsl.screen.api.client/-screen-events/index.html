ScreenEvents

class ScreenEvents

Holds events related to Screens.

A screen being (re)initialized will reset the screen to its default state, therefore reverting all changes a mod developer may have applied to a screen.

The primary entrypoint into a screen is when it is being opened, this is signified by an event before and after initialization of the screen.
