open fun newFileSystemResourcePack(id: Identifier, rootPath: Path, activationType: ResourcePackActivationType): ResourcePack

Creates a new resource pack based on a Path as its root.

If the file system of the given Path is not the operating system's file system then the pack resources will be cached.

Return

a new resource pack instance

Parameters

id

the identifier of the resource pack; its namespace must be the same as the mod ID

rootPath

the root path of this resource pack

activationType

the activation type hint of this resource pack
