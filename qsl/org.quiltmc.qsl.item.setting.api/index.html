The Quilt Item Settings API.

What are net.minecraft.item.Item.Settings?

net.minecraft.item.Item.Settings are ways to add specific traits to net.minecraft.item.Items without creating a subclass. In addition, these traits are applicable to all net.minecraft.item.Items, not just a specific subclass.

This API adds three new settings for items, customDamage, equipmentSlot, and recipeRemainder.

These custom settings make use of the org.quiltmc.qsl.item.setting.api.CustomItemSetting API provided. This API allows mods to specify their own custom settings in an API.
