class EntitySelectorOptionRegistry

Class to allow registration of custom entity selector options.

These are registered with namespaced identifiers to avoid name collisions. These are then converted to a name of the form "namespace_path". Due to limitations in how entity selectors may be named, the character "/" is replaced with "_".
