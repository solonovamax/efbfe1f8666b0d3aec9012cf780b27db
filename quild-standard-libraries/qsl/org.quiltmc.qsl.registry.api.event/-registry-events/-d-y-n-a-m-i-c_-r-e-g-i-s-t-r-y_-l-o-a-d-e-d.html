DYNAMIC_REGISTRY_LOADED

val DYNAMIC_REGISTRY_LOADED: Event<RegistryEvents.DynamicRegistryLoadedCallback>

This event gets triggered when a new DynamicRegistryManager gets created, after it has been filled with the registry entries specified by data packs and after the registries have been frozen.

This event can be used to register callbacks to dynamic registries, or to inspect values.

Important Note: The passed dynamic registry manager might not contain the registry, as this event is invoked for each layer of the combined registry manager, and each layer holds different registries. Use getOptional to prevent crashes.
