The Networking (client-side) API.

For login stage networking see org.quiltmc.qsl.networking.api.client.ClientLoginNetworking. For play stage networking see org.quiltmc.qsl.networking.api.client.ClientPlayNetworking.

For events related to connection to a server see org.quiltmc.qsl.networking.api.client.ClientLoginConnectionEvents for login stage or org.quiltmc.qsl.networking.api.client.ClientPlayConnectionEvents for play stage.

For events related to the ability of a server to receive packets on a channel of a specific name see org.quiltmc.qsl.networking.api.client.C2SPlayChannelEvents.
