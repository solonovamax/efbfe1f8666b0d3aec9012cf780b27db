SurfaceRuleEvents

class SurfaceRuleEvents

Events relating to surface rules.

Modification events like MODIFY_OVERWORLD, MODIFY_NETHER, and MODIFY_THE_END allows to modify the surface rules for the related Vanilla dimensions.
